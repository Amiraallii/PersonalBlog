# Personal Portfolio & Blog Platform

A modern, full-stack personal platform featuring a dynamic blog, portfolio showcase, experience timeline, and a robust admin management system. Built with .NET 9 and React 19.

**Live Demo:** [amirali.me](https://amirali.me)

## Features

### Public User Features
- **Home Page:** Displays the top 3 latest blog posts and the top 3 recent projects.
- **About Me:** A brief biography of the owner with dynamic social links (LinkedIn, Twitter/X, WhatsApp, Email, etc.).
- **Experiences & Projects:** A detailed timeline of work history and projects with an option for users to "Request a Project / Contact".
- **Blog & Comments:** Read blog posts and interact through a nested comment system (Write, Reply, Edit, and Delete own comments).
- **Authentication:** Secure Registration, Login, and Logout functionality.

### Admin Dashboard Features
- **Content Management:** Create, Edit, and Delete blog posts and experience items.
- **Contact Info Management:** Dynamically add, update, or remove social media links and contact channels.

## Tech Stack & Architecture

- **Backend:** .NET 9 Web API
- **Frontend:** React 19.2
- **Database:** Microsoft SQL Server 2026 (Code-First Approach)
- **Architecture:** Clean Architecture (Domain-Driven style)
